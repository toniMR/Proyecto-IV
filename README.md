# Proyecto-IV

## Microservicio de Proyección de Películas

El microservicio proporcionará información sobre las películas que hay en un cine. Mostrará las salas, horas y butacas disponibles.

### Herramientas

- **Framework:** Flask
- **Tests:** unitest y Travis CI
- **Base de datos:** MySQL
